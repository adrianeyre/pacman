## <a name="Install">Installation</a>
* To clone the repo
```shell
$ git clone https://github.com/adrianeyre/pacman
$ cd pacman
```
## <a name="Usage">Useage</a>
* Open `Pacman.sln` in Visual Studio
* From the menu `Test > Run > All Tests`
* Click `Start`
